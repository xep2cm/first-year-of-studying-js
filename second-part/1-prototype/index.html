JavaScript Prototype

Прототипное наследование — один из ключевых аспектов языка JavaScript. Оно позволяет объектам наследовать свойства и методы друг друга, создавая цепочку прототипов (Prototype Chain). Каждый объект имеет ссылку на свой прототип, который используется для поиска свойств и методов, отсутствующих непосредственно в самом объекте.

Что такое Прототип?

  • Каждый объект в JavaScript имеет свойство __proto__, которое ссылается на его прототип.
  • Объекты создаются с использованием конструктора (например, new Object()) или литерала объекта ({...}), причем каждый конструктор также обладает собственным методом .prototype.
  • При обращении к свойствам объекта сначала проверяется наличие этих свойств в самом объекте, и лишь потом идет обращение к прототипу.

Пример:


// Создаем простой объект Person
function Person(name) {
    this.name = name;
}

// Добавляем метод greet в прототип Person
Person.prototype.greet = function() {
    console.log(`Привет! Меня зовут ${this.name}.`);
};

const person = new Person('Иван');
person.greet(); // Привет! Меня зовут Иван.

Заключение

Понимание системы прототипов крайне важно для освоения основ JavaScript и эффективного программирования на этом языке. Именно благодаря этому механизму объекты приобретают гибкость и возможность повторного использования функционала.
