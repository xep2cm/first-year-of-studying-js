Урок 5. JavaScript. Promise.

Что такое Promise?

Promise — это объект, представляющий собой обещание завершения асинхронной операции и её результата. Обещания позволяют лучше организовать код и упростить обработку ошибок в асинхронных операциях.

Основные состояния:

  • pending: начальное состояние ожидания результата.
  • fulfilled: операция успешно выполнена.
  • rejected: произошла ошибка во время выполнения операции.

Методы Promise:

  1. .then(): метод обрабатывает успешное завершение промиса.
  2. .catch(): метод обрабатывает ошибку промиса.
  3. .finally(): выполняется независимо от успеха или отказа промиса.

Пример использования Promise:

// Пример использования Promise

const delay = ms => new Promise(resolve => setTimeout(() => resolve(ms), ms));

delay(2000)
    .then(result => console.log(`Прошло ${result} миллисекунд`))
    .catch(error => console.error('Ошибка:', error))
    .finally(() => console.log('Операция завершена'));
