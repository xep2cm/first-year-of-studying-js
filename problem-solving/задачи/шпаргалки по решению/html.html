Шпаргалка по работе с массивами и объектами в JavaScript

Эти материалы помогут быстро вспомнить ключевые концепции и структуры, необходимые для эффективного взаимодействия с массивами и объектами.

Работа с массивами объектов

Массивы объектов представляют собой наиболее распространённый способ хранения сложных данных:

const users = [
    { id: 1, name: 'Иван', email: 'ivan@example.com' },
    { id: 2, name: 'Марина', email: 'marina@example.com' },
];

Основные операции над массивами объектов

Фильтрация по условию

Отбор записей по какому-либо признаку:

users.filter(user => user.id === 1);

Пример использования:

const filteredUsers = users.filter(user => user.email.includes('@example.com'));
// Результат: [{id: 1, name: 'Иван', email: 'ivan@example.com'}, {id: 2, name: 'Марина', email: 'marina@example.com'}]

Преобразование массива (Map)

Создание нового массива путём преобразования каждого элемента:

users.map(user => user.name);

Пример использования:

const names = users.map(user => user.name);
// Результат: ["Иван", "Марина"]

Агрегирование (Reduce)

Накопление суммарного значения:

users.reduce((sum, user) => sum + user.id, 0);

Пример использования:

const totalIds = users.reduce((sum, user) => sum + user.id, 0);
// Результат: 3

Группировка данных

Разделение данных по группам:

users.reduce((groups, user) => {
    groups[user.gender] ||= []; // ES2022 синтаксис инициализации массива
    groups[user.gender].push(user);
    return groups;
}, {});

Пример использования:

const groupedByGender = users.reduce((groups, user) => {
    groups[user.gender] ||= [];
    groups[user.gender].push(user);
    return groups;
}, {});
// Результат: {"male": [], "female": []}

Поиск уникального набора значений

Извлечение уникальных значений:

const uniqueEmails = [...new Set(users.map(user => user.email))];

Пример использования:

const emails = users.map(user => user.email);
const uniqueEmails = [...new Set(emails)];
// Результат: ["ivan@example.com", "marina@example.com"]

Простой перебор массива

Перебор без изменений:

users.forEach(user => console.log(user.name));

Пример использования:

users.forEach(user => console.log(`${user.name}: ${user.email}`));
// Результат: "Иван: ivan@example.com", "Марина: marina@example.com"

Дополнительные полезные техники

Доступ к глубоким вложениям

Можно обращаться к внутренним ключам объекта:

user.address.city;

Пример использования:

const addressCity = user.address.city;
// Результат: Москва

Определение количества элементов по условию

Подсчёт количества подходящих записей:

users.filter(user => user.id > 1).length;

Пример использования:

const count = users.filter(user => user.id > 1).length;
// Результат: 1

Поиск первого совпадающего элемента

Быстрый поиск первой подходящей записи:

users.find(user => user.id === 1);

Пример использования:

const foundUser = users.find(user => user.id === 1);
// Результат: {id: 1, name: 'Иван', email: 'ivan@example.com'}

Советы по улучшению кода

- Используйте осмысленные и короткие имена переменных. - Пользуйтесь деструктуризацией для удобного извлечения данных: 
const { name, email } = user;
- Объединяйте различные методы для оптимизации кода и повышения производительности. ---
