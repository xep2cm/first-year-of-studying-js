Шпаргалка по основам JavaScript: Числа, Строки, Дата

Числа в JavaScript

JavaScript представляет числа двумя основными способами:

  • Number: обычный числовой тип данных, используемый для всех видов чисел (целых и дробных). Все числа хранятся в виде двойных с плавающей запятой IEEE-754.
  • BigInt: специальный тип для поддержки очень больших целых чисел, которые превышают пределы обычного Number-типа.

Основные свойства и методы чисел:

  • toFixed(n): округляет число до n знаков после запятой.
  • toPrecision(n): выводит число с заданной точностью (общим количеством цифр).
  • Math.random(): возвращает случайное число от 0 до 1.
  • parseFloat(str), parseInt(str): преобразуют строку в число.
let x = 10;
x += 5; // 15
console.log(x.toFixed(2)); // "15.00"

Строки в JavaScript

Строковые литералы в JavaScript создаются несколькими способами:

  • Одинарные кавычки (' ').
  • Двойные кавычки (" ").
  • Шаблонные строки (` `), позволяющие встраивание выражений.

Методы строк:

  • length: свойство, возвращающее длину строки.
  • slice(start, end): извлекает часть строки.
  • split(separator): делит строку на массив по указанному разделителю.
  • trim(): удаляет пробелы в начале и конце строки.
let str = "Привет мир!";
console.log(str.slice(0, 5)); // "Привет"

Работа с датой и временем в JavaScript

Объект Date используется для работы с датой и временем. Конструктор Date принимает различные форматы инициализации:

  • Без аргументов — текущая дата и время.
  • Год, месяц, день, часы, минуты, секунды, миллисекунды.
  • ISO формат даты (например, "2025-08-14").

Основные методы объекта Date:

  • getFullYear(): возвращает полный год.
  • getMonth(): возвращает номер месяца (0—11).
  • getHours(), getMinutes(), getSeconds(): получение часов, минут и секунд соответственно.
  • toLocaleString(locale): форматирует дату согласно указанной локализации.
let now = new Date();
console.log(now.getFullYear()); // Год, например 2025
